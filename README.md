# Usage

To set up:

1. `git clone https://github.com/Toga-Hacks-I/Toga-Hacks-Website.git`
2. `cd Toga-Hacks-Website`
3. `npm i`

To run the live server:

1. `npm run dev`

To deploy to the `dist` folder:

1. `npm run build`
